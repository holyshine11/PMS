당신은 호텔 PMS, 하우스키핑 운영 프로세스, 관리자 권한 설계, 모바일 웹 UX/UI, Spring Boot 기반 시스템 설계에 숙련된 시니어 솔루션 아키텍트입니다.

현재 Hola PMS를 개발 중이며, 룸랙(객실현황) 기능까지는 구현된 상태입니다.
이제 하우스키핑 메뉴 및 기능을 신규 설계하려고 합니다.

[요구사항]
- PMS 관리자에 하우스키핑 메뉴 신설
- 하우스키퍼가 사용할 Mobile Web 화면 필수 구성
- 현재 호텔 > 프로퍼티 권한까지만 구현되어 있음
- 하우스키퍼 권한 관리 메뉴 및 기능 추가 개발 필요
- 권한 부여 로직은 기존 시스템 로직을 최대한 참고
- @Reference(OPERA Cloud) 폴더에 오페라 관리자 화면 캡처가 있음
- 단순 참고가 아니라 OPERA Cloud보다 더 편리하고 직관적인 UI/UX로 설계 필요
- 시간과 룸 상태를 함께 관리할 수 있도록 타임라인, 캘린더, 작업보드 등의 방식 검토 필요
- 기존 룸랙과 자연스럽게 연결되는 구조여야 함

## 룸랙 연계 강조
하우스키핑 기능은 별도 메뉴로도 제공되지만,
기존 룸랙 화면과 연계되어 객실 상태 흐름을 동일 맥락에서 파악할 수 있어야 합니다.
따라서 룸랙 연동 방식과 화면 진입 포인트도 함께 제안해 주세요.

## 모바일 실사용성 강조

모바일웹은 단순 조회 화면이 아니라 실제 하우스키퍼 현장 작업용 화면입니다.
따라서 한 손 조작, 대버튼, 빠른 상태 변경, 최소 입력, 네트워크 불안정 시 대응까지 고려해 주세요.
하우스 키퍼용 프론트 모바일 웹 페이지가 꼭 필요합니다.

## 향후 확장성 반영

향후에는 유지보수(Maintenance), 미니바, 객실 점검(Inspection), 분실물(Lost & Found), 게스트 요청(Request) 기능과도 연계될 수 있도록 확장 가능한 구조를 고려해 주세요.
